## 🎯 Présentation
Youna-v2 est une lettre d'amour numérique qui se déploie en quatre chapitres animés : une bougie que l'on allume, une enveloppe scellée à la cire que l'on ouvre, une lettre manuscrite révélée avec des contrôles de lecture, et une conclusion visuelle. Chaque étape est chorégraphiée avec soin pour offrir une expérience intime et immersive.

---

## ✨ Fonctionnalités
- ✅ Bougie animée avec flamme et fumée (particules)
- ✅ Enveloppe avec sceau de cire à ouvrir
- ✅ Lettre manuscrite déroulante avec contrôles de lecture (play/pause/reprise)
- ✅ Page de conclusion avec animation de particules
- ✅ Musique d'ambiance intégrée (Howler.js)
- ✅ Animations fluides GSAP tout au long du parcours

## 🚀 Installation
```bash
git clone https://github.com/MattiaPARRINELLO/Youna-v2.git
cd Youna-v2
# Ouvrir index.html dans un navigateur ou servir avec un live-server
```

---

## 📖 Utilisation
Ouvrez simplement le fichier `index.html` dans votre navigateur ou utilisez un serveur local (Live Server, etc.). Suivez le parcours interactif : allumez la bougie, ouvrez l'enveloppe, lisez la lettre et découvrez la conclusion. Le son ambiant se lance automatiquement.
